# linktree-profissional
Central de links profissional desenvolvida com HTML5 semântico, CSS3 (Flexbox/Responsividade) e JavaScript dinâmico. Projeto focado em demonstrar organização de informações, manipulação do DOM e lógica de programação para Análise e Suporte Técnico.
